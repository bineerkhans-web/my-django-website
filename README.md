#  My Django Website

A dynamic web application built with **Django 5.2.7**, **Bootstrap 5**, and **SQLite3**.

##  Features
- User authentication  
- File uploads (images)
- Responsive UI with Bootstrap  
- Filtering (Isotope.js) & Sliders (Owl Carousel)  
- Integrated Google Maps

```bash
